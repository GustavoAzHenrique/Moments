Sobre o Moments

"O projeto Moments foi desenvolvido em correlação com o conteúdo e material apresentado pelo canal do YouTube: Matheus Battisti - Hora de Codar. Tem por intuito um ambiente de compartilhamento de momentos através de posts. O projeto foi desenvolvido utilizando linguagem TypeScript com o framework Angular e uma API em AdonisJS, o mvp possui todas as ações de CRUD inseridas."

Compartilhe seus momentos, e receba comentários de quem você mais gosta!
